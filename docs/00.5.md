# Python仮想環境を作ってQiskitをインストールする

量子プログラミングではライブラリ依存が多いため、仮想環境を使うのが安全です。

ここでは Python 標準の `venv` を使います。

(以下を手順を筆者はMac Tahoe 26.3で確認)

---

# ① Python3が入っているか確認

まず確認：

```bash
python3 --version
```

例：

```text
Python 3.11.6
```

が出ればOKです。

---

# ② 仮想環境を作成

次に `quantum_study` という環境を作ります：

```bash
python3 -m venv quantum_study
```

すると次のディレクトリが作られます：

```text
quantum_study/
```

この中に Python が独立してインストールされます。

---

# ③ 仮想環境を有効化（activate）

Mac / Linux：

```bash
source quantum_study/bin/activate
```

Windows（PowerShell）：

```powershell
quantum_study\Scripts\Activate.ps1
```

成功するとプロンプトが変わります：

```bash
(quantum_study) $
```

これが出れば仮想環境に入っています。

---

# ④ pip を最新版に更新（推奨）

```bash
pip install --upgrade pip
```

---

# ⑤ Qiskit をインストール

基本セット：

```bash
pip install qiskit
```

シミュレータも使う場合：

```bash
pip install qiskit-aer
```

GPUなしならここまででOKです。

---

# ⑥ インストール確認

Pythonを起動：

```bash
python
```

そして：

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(1,1)
print(qc)
```

エラーが出なければ成功です。

---

# ⑦ 仮想環境を終了する

作業が終わったら：

```bash
deactivate
```

---

# ⑧ 次回からの起動方法

毎回これだけ：

```bash
source quantum_study/bin/activate
```

# 1. 量子もつれとは何か？

まず直感的に言うと：

```
2つの量子ビットが独立でなくなる状態
```

です。

普通の2ビットなら：

```
00
01
10
11
```

のどれかです。

また重ね合わせなら：

```
(|0⟩ + |1⟩)/√2
```

のように書けます。

しかし量子もつれでは：

```
2つの量子ビットを別々に説明できない
```

状態が現れます。

これが量子計算の本質です。

---

# 2. 例：もっとも有名なもつれ状態（Bell状態）

代表例：

```
(|00⟩ + |11⟩)/√2
```

これは：

```
両方0
または
両方1
```

が同時に存在する状態です。

ここで重要なのは：

```
片方だけでは状態が決まらない
```

という点です。

---

# 3. 古典計算との違い

例えば古典的な乱数なら：

```
00 または 11
```

は表現できます。

しかし：

```
(|00⟩ + |11⟩)/√2
```

は違います。

なぜなら：

```
測定するまで決まっていない
```

からです。

さらに：

```
片方を観測するともう片方も決まる
```

という相関が発生します。

これが：

```
量子もつれ
```

です。

---

# 4. Bell状態を作ってみる

では実際に回路を書きます。

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(2, 2)

qc.h(0)
qc.cx(0, 1)

qc.measure([0,1], [0,1])

print(qc.draw())
```

ここで重要なのは：

```
qc.h(0)
```

と

```
qc.cx(0,1)
```

です。

---

# 5. Hゲートの役割

まず：

```python
qc.h(0)
```

によって：

```
|0⟩ → (|0⟩ + |1⟩)/√2
```

になります。

つまり状態は：

```
(|00⟩ + |10⟩)/√2
```

になります。

---

# 6. CNOTゲートとは何か？

次に：

```python
qc.cx(0,1)
```

です。

これは：

```
制御ビットが1なら
ターゲットビットを反転
```

するゲートです。

つまり：

```
10 → 11
```

になります。

結果として：

```
(|00⟩ + |11⟩)/√2
```

になります。

これが：

```
Bell状態
```

です。

---

# 7. なぜこれが特別なのか？

ここが最重要ポイントです。

次の状態：

```
(|00⟩ + |11⟩)/√2
```

は：

```
1つ目の量子ビット
×
2つ目の量子ビット
```

の積として書けません。

つまり：

```
独立な2つの量子ビットではない
```

ということです。

これを：

```
非分離状態
```

と呼びます。

---

# 8. 実行結果を確認する

この回路を実行すると：

```
00
```

または

```
11
```

だけが出ます。

```
01
10
```

は出ません。

つまり：

```
必ず同じ値になる
```

という相関が生まれています。

しかしこれは単なる乱数ではありません。

なぜなら：

```
測定するまで値は存在しない
```

からです。

---

# 9. なぜ量子もつれが重要なのか？

量子もつれは：

```
量子計算の計算能力の源
```

です。

例えば：

* 量子テレポーテーション
* Shorのアルゴリズム
* Grover探索
* 量子暗号

すべて：

```
もつれ
```

を使います。

逆に言うと：

```
もつれがなければ量子計算は強くならない
```

とも言えます。

---

# 10. 開発者としての理解ポイント

ここまでの話を整理すると：

重ね合わせ：

```
1ビットの拡張
```

もつれ：

```
複数ビットの結合
```

です。

つまり：

```
状態空間そのものが拡張される
```

ということです。

## コラム：CNOTとBellという名前の意味

ここで登場した：

    CNOT
    Bell状態

という名前にも意味があります。

まず CNOT は：

    Controlled NOT

の略です。

NOTゲートは、古典コンピュータでいう「0を1に、1を0に反転する」操作です。

CNOTでは、それに：

    Controlled

つまり「制御された」という意味が付きます。

そのため CNOT は：

    制御ビットが1のときだけ
    ターゲットビットにNOTをかける

というゲートです。

名前そのものが、動作をかなり直接表しています。

一方、Bell状態の Bell は、人名です。

これは物理学者：

    John Stewart Bell

に由来します。

Bellは、量子もつれが単なる「隠れた情報」では説明できないことを示す重要な考え方を提示しました。

その文脈で、2つの量子ビットが強くもつれた代表的な状態を：

    Bell状態

と呼びます。

今回出てきた：

    (|00⟩ + |11⟩)/√2

は、その代表例です。

つまりまとめると：

| 名前 | 由来 | 意味 |
|---|---|---|
| CNOT | Controlled NOT | 条件付きで反転するゲート |
| Bell状態 | John Stewart Bell | 量子もつれを代表する状態 |

CNOTは「操作の名前」、
Bell状態は「もつれた結果の名前」

と考えると分かりやすいです。

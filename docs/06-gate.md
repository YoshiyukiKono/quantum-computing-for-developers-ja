# 1. 振幅だけではなく「位相」がある

これまで量子状態を：

$$
|\psi\rangle = \alpha|0\rangle + \beta|1\rangle
$$

と書いてきました。

重要なのは：

$$
\alpha,\beta は実数ではなく複素数
$$

だということです。

つまり：

```
確率 = 振幅の絶対値²
```

ですが、

```
干渉 = 位相の差
```

によって起きます。

例：

```
|0⟩ + |1⟩
```

と

```
|0⟩ − |1⟩
```

は確率としては同じでも、

**干渉結果が変わります。**

ここが量子計算の核心です。

---

# 2. 位相とは何か？

位相とは：

> 振幅の符号や角度の違い

です。

例：

### 状態A

```
|ψ⟩ = |0⟩ + |1⟩
```

### 状態B

```
|ψ⟩ = |0⟩ − |1⟩
```

違いは：

```
|1⟩の位相が反転している
```

だけです。

しかしこの違いが：

```
干渉結果を変える
```

のです。

---

# 3. 位相操作をする量子ゲート

位相を操作する代表的なゲート：

## Zゲート

```
|0⟩ → |0⟩
|1⟩ → −|1⟩
```

つまり：

```
|1⟩の位相だけ反転
```

します。

Qiskitでは：

```python
qc.z(0)
```

と書きます。

---

# 4. なぜ位相操作が干渉を生むのか？

ここが量子アルゴリズムの核心です。

例：

まず Hadamard を適用：

```
|0⟩ → |0⟩ + |1⟩
```

次に Z を適用：

```
|0⟩ + |1⟩
↓
|0⟩ − |1⟩
```

さらにもう一度 Hadamard：

すると：

```
|0⟩ − |1⟩
↓
|1⟩
```

になります。

つまり：

```
位相操作だけで測定結果を変えられる
```

のです。

これが：

> 干渉による確率操作

です。

---

# 5. 位相操作＝答えだけを反転する

量子探索アルゴリズムの基本アイデア：

```
正解だけ位相を反転する
```

つまり：

```
|x⟩ → (-1)^f(x)|x⟩
```

ここで：

```
f(x)=1 なら位相反転
f(x)=0 ならそのまま
```

です。

これを：

```
オラクル（oracle）
```

と呼びます。

---

# 6. Groverアルゴリズムとは何か？

Groverアルゴリズムは：

```
正解の確率だけ増やす
```

アルゴリズムです。

流れ：

① 全状態を重ね合わせ
② 正解の位相を反転
③ 全体に干渉操作
④ 正解確率が増える

これを繰り返します。

---

# 7. 2量子ビット版 Grover の最小例

今回は：

```
4個の候補から1つ探す
```

問題を解きます。

つまり：

```
|00⟩
|01⟩
|10⟩
|11⟩
```

の中から：

```
|11⟩ を探す
```

とします。

---

# 8. Qiskitで最小Groverを書く

まず回路を作ります：

```python
from qiskit import QuantumCircuit
from qiskit_aer import AerSimulator
from qiskit.visualization import plot_histogram
from qiskit import transpile

qc = QuantumCircuit(2,2)
```

---

## Step1：重ね合わせを作る

```python
qc.h([0,1])
```

状態：

```
(|00⟩ + |01⟩ + |10⟩ + |11⟩)/2
```

---

## Step2：正解の位相を反転

今回は：

```
|11⟩
```

だけ反転します。

```python
qc.cz(0,1)
```

すると：

```
|11⟩ → −|11⟩
```

になります。

---

## Step3：干渉操作（拡散演算）

次に：

```python
qc.h([0,1])
qc.z([0,1])
qc.cz(0,1)
qc.h([0,1])
```

これが：

```
平均からの反転
```

と呼ばれる操作です。

Groverの核心です。

---

## Step4：測定

```python
qc.measure([0,1],[0,1])
```

---

## Step5：実行

```python
sim = AerSimulator()
compiled = transpile(qc, sim)
result = sim.run(compiled, shots=1000).result()

counts = result.get_counts()
print(counts)
```

結果：

```
'11' が高確率で出る
```

はずです。

---

# 9. 何が起きているのか？

整理すると：

### 最初

```
全候補が同じ確率
```

### 次

```
正解だけ位相反転
```

### 次

```
干渉によって正解だけ強調
```

つまり：

```
振幅を直接変えたわけではない
```

のです。

やったのは：

```
位相を変えただけ
```

です。

それなのに：

```
確率が変わる
```

ここが量子計算の魔法です✨

# はじめに

前回（第6回）では：

> 位相操作が干渉を生む

という話をしました。

ここで登場したのが：

**位相（phase）**

でした。

しかし実は：

> 位相は「複素数」が分からないと本質的に理解できない

概念です。

そこで今回は：

**量子コンピューティングのための複素数入門**

を扱います。

数学としてではなく：

> 量子回路を理解するための道具

として説明します。

---

# 1. なぜ量子コンピュータに複素数が出てくるのか？

古典コンピュータでは：

```text
0 or 1
```

だけを扱います。

確率モデルでも：

```text
0.2
0.7
0.9
```

のような**実数**を扱います。

しかし量子コンピュータでは：

```text
0.6 + 0.8i
```

のような数が出てきます。

これは：

> 状態の確率ではなく「振幅」

だからです。

重要な関係：

```
確率 = 振幅²（正確には絶対値²）
```

です。

つまり：

振幅は確率そのものではありません。

---

# 2. そもそも複素数とは何か？

複素数とは：

```
a + bi
```

という形の数です。

ここで：

* a：実数部分
* b：虚数部分
* i：√−1

です。

例えば：

```
3 + 2i
```

や

```
−1 + 0.5i
```

など。

---

# 3. なぜ √−1 が必要なのか？

普通に考えると：

```
√−1 は存在しない
```

ように思えます。

しかし数学では：

```
i = √−1
```

という新しい数を導入します。

すると：

例えば

```
x² + 1 = 0
```

という方程式が解けます。

つまり：

複素数とは

> 方程式を自然に解けるようにするために導入された数

です。

そして物理学では：

この数が「現実の世界」を非常にうまく表現することが分かりました。

その代表例が：

量子力学

です。

---

# 4. 複素数を図で理解する

複素数は：

数直線ではなく

**平面**

として表現できます。

普通の数：

```
3
```

は横方向だけでした。

複素数：

```
3 + 2i
```

は：

* 横：3
* 縦：2

になります。

つまり：

```
複素数 = 平面上の点
```

です。

これを：

**複素平面**

と呼びます。

---

# 5. 複素数の本当に重要な意味（長さと角度）

複素数は：

```
位置
```

として見ることもできますが、

量子コンピューティングでは：

```
長さ + 角度
```

として見るのが重要です。

例えば：

```
1 + 0i
```

は：

長さ = 1
角度 = 0°

です。

一方：

```
0 + 1i
```

は：

長さ = 1
角度 = 90°

になります。

つまり：

複素数とは

> 長さと回転を同時に表す数

なのです。

---

# 6. 位相とは「角度」のこと

ここで重要なポイント：

量子状態：

```
|ψ⟩ = α|0⟩ + β|1⟩
```

の

```
α
β
```

は複素数でした。

つまり：

それぞれが

```
長さ + 角度
```

を持っています。

この：

角度

が

> 位相（phase）

です。

---

# 7. なぜ位相が重要なのか？

量子コンピュータでは：

確率は

```
|α|²
```

で決まります。

ここに角度は出てきません。

つまり：

```
|1⟩
```

と

```
−|1⟩
```

は同じ確率です。

しかし：

干渉では違いが出ます。

例：

```
|0⟩ + |1⟩
```

と

```
|0⟩ − |1⟩
```

は

測定確率は同じですが、

次に Hadamard をかけると：

結果が変わります。

つまり：

> 位相は測定では見えないが計算では効く

のです。

ここが量子アルゴリズムの核心です。

---

# 8. 位相＝回転操作

複素数は：

回転として理解できます。

例えば：

```
1
```

に

```
i
```

を掛けると：

```
i
```

になります。

これは：

90°回転

です。

さらにもう一度掛けると：

```
−1
```

になります。

つまり：

さらに90°回転です。

まとめると：

```
複素数の掛け算 = 回転
```

です。

---

# 9. 量子ゲートは回転装置である

量子ゲートとは何か？

実は：

```
状態ベクトルを回転させる操作
```

です。

例えば：

Zゲート：

```
|0⟩ → |0⟩
|1⟩ → −|1⟩
```

これは：

```
|1⟩だけ180°回転
```

しています。

つまり：

Zゲートは

**位相回転ゲート**

です。

---

# 10. Hadamardゲートも回転

Hadamard：

```
|0⟩ → |0⟩ + |1⟩
```

これは：

状態空間の中で

回転

を起こしています。

量子回路とは：

```
回転の組み合わせ
```

なのです。

---

# 11. なぜ実数ではダメなのか？

重要な疑問：

なぜ複素数が必要なのか？

答え：

```
干渉を作るため
```

です。

もし振幅が実数だけなら：

できる操作は

```
符号反転
```

程度に制限されます。

しかし複素数なら：

```
任意の角度回転
```

ができます。

つまり：

```
干渉を自由に設計できる
```

のです。

これが：

量子アルゴリズムを可能にしています。

---

# 12. Groverアルゴリズムとの関係

第6回で扱った：

Groverアルゴリズム

では：

① 正解の位相を反転
② 全体を干渉させる

という操作を行いました。

これは実際には：

```
複素数の回転
```

を利用しています。

つまり：

Groverとは

> 位相を設計するアルゴリズム

なのです。

# コラム：複素数はなぜ発明されたのか？そしてなぜ役に立つのか

ここまで読んで、多くの人がこう思ったはずです：

> √−1 なんて本当に必要なの？

実は複素数は：

**最初は誰も信じていなかった数**

でした。

しかし現在では：

* 電気工学
* 信号処理
* 制御理論
* 量子力学
* 量子コンピューティング

のすべてに不可欠です。

なぜそうなったのかを見てみましょう。

---

# 1. 複素数は「方程式を解くため」に生まれた

複素数の起源は16世紀です。

三次方程式：

$$
x^3 + ax + b = 0
$$

を解こうとした数学者たちが、

途中で

$$
\sqrt{-1}
$$

に出会いました。

代表的な人物：

* Gerolamo Cardano
* Rafael Bombelli

彼らは最初こう考えました：

> 計算途中に変な数が出るけど最後は消えるから問題ない

つまり：

```text
計算途中だけ現れる謎の数
```

だったのです。

---

# 2. 「虚数」は長い間疑われていた

当時の数学者はこう呼びました：

```text
imaginary number（想像上の数）
```

つまり：

> 本当に存在しない数

だと思われていました。

しかし18世紀になると、

Leonhard Euler

が革命的な式を発見します：

$$
e^{i\theta} = \cos\theta + i\sin\theta
$$

これは：

**指数関数と三角関数がつながる**

という驚くべき結果でした。

ここで初めて：

```text
複素数 = 回転
```

という意味が見えてきました。

---

# 3. 複素数は「平面の回転」を表す数だった

さらに19世紀、

Carl Friedrich Gauss

が示しました：

> 複素数は平面上の点として理解できる

これによって：

```text
意味不明の数
```

だった虚数は：

```text
幾何学的に意味を持つ数
```

になりました。

ここから：

複素数は数学の中心的存在になります。

---

# 4. 電気工学が複素数を「現実の道具」に変えた

複素数が本当に重要になったのは、

19世紀後半の電気工学です。

交流電流は：

$$
V(t) = V_0 \sin(\omega t)
$$

のように振動します。

これを扱うとき：

複素数を使うと計算が劇的に簡単になります。

例えば：

* フーリエ変換
* フィルタ設計
* 通信理論
* 制御理論

すべて複素数で書かれています。

つまり：

> 現代の電子機器は複素数で動いている

と言っても過言ではありません。

---

# 5. そして量子力学が決定的だった

20世紀に入ると、

Erwin Schrödinger

が導入した波動関数：

$$
\psi(x,t)
$$

は複素数でした。

ここで重要なのは：

```text
自然そのものが複素数で動いている
```

という事実です。

量子状態は：

$$
|\psi\rangle = \alpha|0\rangle + \beta|1\rangle
$$

のように書かれますが、

この

$$
\alpha,\beta
$$

が複素数である理由は：

数学の都合ではなく

**物理の要請**

です。

---

# 6. 量子コンピュータは「複素数を直接計算する装置」

古典コンピュータ：

```text
ビットを操作する
```

量子コンピュータ：

```text
複素振幅を操作する
```

と言えます。

つまり：

量子ゲートとは

```text
複素平面上の回転操作
```

です。

そして：

干渉とは

```text
回転の差
```

です。

さらに：

Groverアルゴリズムとは

```text
位相回転を設計するアルゴリズム
```

です。

---

複素数の歴史は面白い教訓を示しています：

16世紀：

```text
意味不明の数
```

```text
便利な計算道具
```

```text

```


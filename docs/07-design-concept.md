# 1. 量子回路設計とは何をすることか？

古典プログラム：

```
if x == 3:
    return True
```

量子プログラム：

```
|x⟩ → f(x) を位相として埋め込む
```

つまり：

量子計算では

> 条件分岐ではなく「位相操作」を設計する

のが本質です。

量子回路設計とは：

* 問題を
* 位相変化として表現し
* 干渉で答えを強調する

ことです。

---

# 2. オラクルとは何か？

Groverアルゴリズムの中心は：

> 正解だけに印をつける操作

です。

これを：

**オラクル（oracle）**

と呼びます。

数学的には：

```
f(x) = 1 のとき位相反転
f(x) = 0 のときそのまま
```

回路としては：

```
|x⟩ → (-1)^f(x) |x⟩
```

つまり：

正解だけ符号を反転します。

---

# 3. 2量子ビットのオラクルを作る

例えば：

```
|11⟩だけ正解
```

としたい場合。

やること：

```
|11⟩だけ位相反転
```

これは：

```
CZゲート
```

で実現できます。

回路：

```
H ───●───H
     │
H ───Z───H
```

ここで：

```
|11⟩ → -|11⟩
```

になります。

つまり：

**条件付き位相反転**

＝

**オラクル**

です。

---

# 4. 任意の状態に印をつける方法

例えば：

```
|01⟩だけ反転したい
```

場合。

手順：

① |01⟩ を |11⟩ に変換
② CZを適用
③ 元に戻す

回路：

```
X(0)
CZ
X(0)
```

これだけです。

つまり：

> 任意の状態に対するオラクルは作れる

のです ✨

# 5. 多量子ビットGroverの考え方

1量子ビット：

```
50% → 100%
```

2量子ビット：

```
25% → 約100%
```

3量子ビット：

```
12.5% → 約100%
```

一般には：

```
N個の候補
```

から：

```
√N 回の反復
```

で答えが見つかります。

つまり：

```
2^n → 2^(n/2)
```

の計算量になります。

これが：

量子探索高速化

です。

# 6. Grover回路の構造（一般形）

Groverは次の2ステップの繰り返しです：

① オラクル
② 拡散演算子（diffusion operator）

構造：

```
初期化
↓
Hゲート
↓
[オラクル]
↓
[平均反転]
↓
繰り返し
↓
測定
```

つまり：

```
位相反転
↓
振幅増幅
```

のループです。

---

# 7. 拡散演算子とは何をしているのか？

拡散演算子は：

> 平均値を中心に反転する

操作です。

イメージ：

```
平均との差を反転
```

つまり：

```
小さい確率 → さらに小さく
大きい確率 → さらに大きく
```

なります。

これが：

干渉による確率増幅

です。

---

# 8. 回路をどう分解するか？

量子回路設計の基本戦略は：

① 解きたい問題を書く
② 条件を論理式にする
③ 位相反転として実装する

例えば：

```
x1 AND x2 AND x3
```

なら：

```
CCZゲート
```

になります。

つまり：

古典論理

↓

量子位相操作

へ変換するのが設計です。

---

# 9. 多制御ゲートはどう作る？

例えば：

```
|111⟩だけ反転
```

なら：

```
CCZ
```

を使います。

さらに増えると：

```
CCCZ
CCCCZ
```

になります。

実際の量子回路では：

これらは分解されます：

```
CNOT
H
T
S
```

などの基本ゲートへ。

つまり：

量子回路設計とは：

> 高レベル回路を書く → 低レベルゲートへ分解する

作業です。

# 10. 実用アルゴリズムへの拡張方法

Groverは探索問題でした。

しかし設計パターンは：

多くのアルゴリズムに共通します。

例：

* SAT問題
* 最適化問題
* 経路探索
* 機械学習
* 暗号解析

すべて：

```
条件を位相に変換
↓
干渉で増幅
```

という構造を持ちます。

つまり：

オラクル設計＝量子アルゴリズム設計

です。

# 11. Python（Qiskit）で多量子ビットGroverを書く

例：

2量子ビットGrover

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(2)

# 初期化
qc.h([0,1])

# オラクル（|11⟩反転）
qc.cz(0,1)

# diffusion
qc.h([0,1])
qc.x([0,1])
qc.cz(0,1)
qc.x([0,1])
qc.h([0,1])

qc.draw("text")
```

構造がそのまま：

```
oracle
+
diffusion
```

になっているのが確認できます。

ここが理解できれば：

自分で回路を書けます。

# 1. 量子機械学習（QML）とは何か？

量子機械学習（Quantum Machine Learning）とは：

> 量子回路を使ってデータを学習する技術

です。

基本構造：

```text
入力データ
↓
量子状態へ変換
↓
量子回路で変換
↓
測定
↓
損失関数
↓
最適化
```

つまり：

VQAの一種

です。

---

# 2. なぜ量子機械学習が注目されるのか？

理由は主に3つあります：

① 高次元特徴空間を自然に扱える
② 干渉を利用した非線形変換
③ カーネル計算の高速化の可能性

つまり：

```text
量子状態 = 高次元ベクトル
```

として扱えることが重要です。

---

# 3. Feature Mapとは何か？

Feature Mapとは：

> 古典データを量子状態に変換する方法

です。

例：

入力：

```text
x = (x₁, x₂)
```

量子状態：

```text
|ψ(x)⟩
```

に変換します。

一般形：

```text
|ψ(x)⟩ = U(x)|0⟩
```

ここで：

U(x)

はデータ依存ユニタリです。

---

# 4. 典型的なFeature Mapの例

代表例：

角度エンコーディング

例：

```text
x → RY(x)
```

つまり：

```text
|0⟩ → RY(x)|0⟩
```

です。

複数量子ビットなら：

```text
RY(x₁)
RY(x₂)
RY(x₃)
```

のように適用できます。

---

# 5. なぜFeature Mapが重要なのか？

理由：

特徴空間を決めるから

です。

古典機械学習でも：

```text
線形分類
```

か

```text
カーネル分類
```

かは：

特徴写像で決まります。

量子でも同じです。

---

# 6. 量子カーネル法とは何か？

量子カーネルとは：

次の量を計算する方法です：

$$
K(x, x') = |\langle \psi(x) | \psi(x') \rangle|^2
$$

意味：

2つのデータの類似度

です。

これは：

SVM（サポートベクターマシン）

で使われます。

---

# 7. なぜ量子カーネルが強力なのか？

理由：

古典では計算困難な特徴空間

を使える可能性があるからです。

直感：

```text
古典特徴空間 → 低次元
量子特徴空間 → 指数次元
```

つまり：

より複雑な分類境界

を表現できます。

---

# 8. 量子ニューラルネット（QNN）とは何か？

QNNとは：

> パラメータ付き量子回路を使った学習モデル

です。

構造：

```text
Feature Map
↓
Parameterized Circuit
↓
Measurement
↓
Loss
↓
Optimization
```

これは：

完全にVQA構造

です。

---

# 9. QNNの数式表現

出力：

$$
f(x, \theta) = \langle 0 | U^\dagger(x, \theta) M U(x, \theta) | 0 \rangle
$$

ここで：

| 記号 | 意味      |
| -- | ------- |
| x  | 入力      |
| θ  | 学習パラメータ |
| M  | 測定演算子   |

つまり：

```text
測定結果 = モデル出力
```

です。

---

# 10. 古典ニューラルネットとの対応関係

対応関係：

| 損失関数  | 期待値  |

量子版ニューラルネット

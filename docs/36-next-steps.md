# 2. 必要な数学と物理

量子コンピューティングは難しい数学が必要だと思われがちですが、

実際には段階があります。

---

## レベル1（実装者向け）

まず必要なのは：

* ベクトル
* 行列
* 複素数
* 確率

だけです。

特に重要なのは：

```
状態 = ベクトル
ゲート = 行列
測定 = 確率
```

という理解です。

これだけで：

* Grover
* QAOA
* Variational circuits

まで扱えます。

---

## レベル2（アルゴリズム設計者向け）

次に必要になるのは：

* 固有値分解
* ユニタリ行列
* テンソル積
* フーリエ変換

ここまで来ると：

* Shor
* HHL
* Phase estimation

が理解できます。

---

## レベル3（研究者向け）

さらに進むと：

* ヒルベルト空間
* 線形作用素
* スペクトル理論
* 量子力学の公理

が必要になります。

ただし開発者なら：

ここまで必須ではありません。
